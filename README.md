# Moulie Malhotra - Portfolio

A modern, responsive portfolio website built with Next.js, TypeScript, and Framer Motion.

## 🚀 Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Responsive**: Works perfectly on all devices
- **Interactive**: Hover effects, smooth scrolling, and engaging animations
- **Fast**: Optimized for performance with Next.js
- **Accessible**: Built with accessibility best practices

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Radix UI

## 📦 Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/yourusername/moulie-portfolio.git
cd moulie-portfolio
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🖼️ Adding Certificate Images

To add your certificate images:

1. Create a `public/certificates/` folder
2. Add your certificate images with these names:
   - `data-science-cert.jpg`
   - `sql-cert.jpg`
   - `python-cert.jpg`
   - `cpp-cert.jpg`
   - `c-cert.jpg`

## 🚀 Deployment

### Deploy to Vercel (Recommended)
1. Push your code to GitHub
2. Connect your GitHub repo to Vercel
3. Deploy automatically on every push

### Deploy to Netlify
1. Build the project: `npm run build`
2. Deploy the `out` folder to Netlify

### Deploy to GitHub Pages
1. Enable GitHub Pages in your repo settings
2. Push your code and it will auto-deploy

## 📝 Customization

- Update personal information in each component
- Modify colors in `tailwind.config.ts`
- Add/remove sections as needed
- Update social media links in the hero section
